# 抖音二创片单查询网站

聚合抖音合作视频平台的可二创影视剧片单，方便创作者快速查询哪些作品可以进行二次创作。

## 合作平台

| 平台 | APP内搜索关键词 | 片单名称 |
|------|----------------|---------|
| 爱奇艺 | 爱奇艺影视内容精选 | 爱奇艺影视内容精选 |
| 优酷 | 优酷视频影视精选 | 优酷视频影视精选 |
| 芒果TV | 芒果必看精选片单 | 芒果必看精选片单 |
| 搜狐视频 | 抖音片单 | 抖音片单 |

## 数据来源

片单数据来自各平台 APP 内的官方二创片单页面，由 AI Agent 通过 iPhone Mirroring 爬取后以 PR 形式提交。

数据格式见 [data/SCHEMA.md](data/SCHEMA.md)。

## 技术栈

- Astro（静态站生成）
- 数据：`data/*.json`（每平台一个文件）
- 部署：GitHub Pages

## 创作要求（抖音官方）

- 不得对长视频原片进行拆条处理，需创作有二次加工的原创内容（解说/杂谈/混剪等）
- 视频内容不得替代原片，不得歪曲、篡改长视频原意
- 视频内容不得违法违规、违反公序良俗、侵害他人肖像权/名誉权等合法权益
- 建议创作视频时长在 4 分钟以内
- 建议添加内容对应的影视综艺标签
